# Amazon Storage
###### 22<sup>nd</sup> March 2026 - Puneet Gavri

AWS storage provides scalable, durable, and secure cloud storage services, including object (S3), block (EBS), and file (EFS/FSx) storage, along with hybrid options (Storage Gateway). These services support diverse needs from high-performance computing to archiving, featuring pay-as-you-go pricing and robust data protection.
